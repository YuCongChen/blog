位运算

位运算是一种使用二进制数来进行计算的计算方式

位运算分为以下几种：

  • 按位与 (&)
  • 按位或 (|)
  • 按位异或 (^)
  • 按位取反 (~)
  • 左移 (<<)
  • 右移 (>>)

按位与 (&)

两个二进制数进行运算，当两个操作数为都为1时结果为1，否则结果为0；

例如：a = 1010; b = 0110; a & b = 0010;

按位或 (|)

两个二进制数进行运算，当两个操作数都为0时结果为0，否则结果为1

例如：a = 1010; b = 0110; a | b = 1110;

按位异或 (^)

两个二进制数进行运算，当两个操作数不同时结果为1，否则结果为0

例如：a = 1010; b = 0110; a | b = 1100;

按位取反 (~)

按位取反只对一个二进制数进行运算，将该二进制数的1转为0，0转为1。按位取反会涉及到二位数的正负数问题，将一个十进制数进行按位取反运算的公式是：a = -(a + 1)

左移运算 (<<)

将一个二进制数左移指定的位数，左侧溢出的数将被舍弃，右侧空余的位置使用0补充。将一个数左移几次实际上就是将该数乘以几次2。1010 << 2 = 1000

右移运算 (>>)

将一个二进制数右移指定的位数，右侧溢出的数将被舍弃，左侧空余的位置使用0补充。将一个数右移几次实际上就是将该数除以几次2。1010 >> 2 = 0010
